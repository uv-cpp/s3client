Detailed Description

S3 Client inteface.

Implements some of the S3 actions documented here: https://docs.aws.amazon.com/AmazonS3/latest/API/API_Operations_Amazon_Simple_Storage_Service.html Also includes higher level methods to simplify access to API.

When sending requests, the fields specified as URL prameters are sent as part of the request, but in most cases the optional parameters sent as HTTP header fields are not sent along, it is therefore required to explicitly pass them in the headers parameter.

E.g. the ListObjectV2 request supports HTTP headers for specifying the payer and owner of the listed objects.

s3.ListObjectsV2(bucket, config, {
{"x-amz-request-payer", "RequestPayer"},
{"x-amz-expected-bucket-owner", "ExpectedBucketOwner"}
});
