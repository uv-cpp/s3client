Detailed Description

Representaion of an XML tree as {path to text element, text element} tuples stored into map object. E.g.

<tag1>
<tag1_1>
text 1 1
</tat1_1>
<tag1_2>
text 1 2
</tag1_2>
</tag1>

is represented as

unordered_map<string, string> rec {
{"/tag1/tag1_1", "text 1 1"},
{"/tag1/tag1_2", "text 1 2"}
};
