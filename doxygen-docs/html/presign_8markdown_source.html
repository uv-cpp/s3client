1 # Presign
2 
3 All requests can be issued through standard web clients such as `curl` specifying a pre-signed url on
4 the command line.
5 
6 ## Example: Upload object
7 
8 ### Presign URL
9 Specify method, bucket name and key and expiration time of 10k seconds.
10 
11 ```
12 export PRESIGN=`build/s3-presign -a $S3CLIENT_TEST_ACCESS -s $S3CLIENT_TEST_SECRET -e $S3CLIENT_TEST_URL -m "PUT" -x 10000 -b buck -k xxx`
13 ```
14 ### Upload object
15 Invoke `curl` and specify custom metadata field by adding an `x-amz-meta-` header.
16 ```
17 curl $PRESIGN -X PUT -F "file=@CMakeCache.txt" -H "x-amz-meta-mymeta:123"
18 ```
19 ### Verify upload
20 Invoke *HEAD* operation on uploaded object and read metadata.
21 
22 ```
23 build/s3-client -a $S3CLIENT_TEST_ACCESS -s $S3CLIENT_TEST_SECRET -e $S3CLIENT_TEST_URL -b buck -k xxx -m HEAD
24 ```
